# 光譜之間：做陳明的夥伴
根據我們對自閉症學術文獻的研究，我們開發了一款迷你沈浸式對話遊戲—「光譜之間：做陳明的夥伴」。

## 遊玩方式

### 下載遊戲文件
```Git
git clone https://github.com/skywalker-lt/in-spectrum.git --recursive
```

### 安裝遊戲
```Zsh
cd in-spectrum
npm install
npm start
```

### 開始遊戲
前往Safari或Google Chrome輸入`http://localhost:3000`

目前由於經費限制，我們只能提供本地自定義安裝包，未來一個月內我們會開設伺服器及CDN，屆時任何人都可訪問in-spectrum.org即刻開啟遊戲！
